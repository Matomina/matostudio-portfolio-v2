# Stratégie de contenu — MatoStudio Portfolio V2

## Objectif

Définir les contenus des deux portfolios avant de construire les sections finales.

---

## 1. Portfolio freelance MatoStudio

### Cible

Clients, indépendants, petites entreprises, associations, artisans, services locaux ou porteurs de projet qui ont besoin d’un site professionnel.

### Objectif du visiteur

Comprendre rapidement :

- qui est MatoStudio ;
- ce qui est proposé ;
- pourquoi faire confiance ;
- combien cela peut coûter ;
- comment prendre contact.

### Objectif business

Transformer le visiteur en prospect qualifié.

---

## Structure cible freelance

### Hero

Objectif : expliquer la promesse en moins de 5 secondes.

Contenu attendu :

- headline premium ;
- sous-titre orienté résultat ;
- CTA principal ;
- CTA secondaire ;
- preuve de sérieux.

### Services

Objectif : montrer ce que MatoStudio peut produire.

Services cibles :

- site vitrine ;
- landing page ;
- refonte ;
- portfolio ;
- intégration responsive ;
- optimisation conversion ;
- accompagnement technique.

### Process

Objectif : rassurer le client.

Étapes possibles :

1. Analyse du besoin
2. Structure et contenu
3. Design et intégration
4. Optimisation responsive
5. Mise en ligne
6. Suivi

### Tarifs / offres

Objectif : donner un repère clair sans enfermer l’offre.

Approche recommandée :

- offre starter ;
- offre business ;
- offre premium ;
- devis personnalisé.

### Projets

Objectif : prouver la capacité de réalisation.

Projets prioritaires :

- MangaBook ;
- AS Transports ;
- LAVEOO ;
- portfolio existant ;
- futurs projets clients.

### Contact

Objectif : réduire la friction.

Informations :

- téléphone ;
- email ;
- formulaire ;
- GitHub ou LinkedIn si pertinent ;
- disponibilité.

---

## 2. Portfolio emploi / entreprise

### Cible

Recruteurs, entreprises, responsables techniques, organismes de formation, partenaires professionnels.

### Objectif du visiteur

Comprendre rapidement :

- ton profil ;
- ton niveau technique ;
- tes projets ;
- ta progression ;
- ta capacité à rejoindre une équipe.

### Objectif carrière

Obtenir des opportunités :

- emploi ;
- alternance/stage si pertinent ;
- mission ;
- immersion ;
- recommandation ;
- réseau professionnel.

---

## Structure cible emploi

### Hero

Objectif : te positionner comme développeur web professionnel.

Contenu attendu :

- titre clair ;
- stack principale ;
- disponibilité ou objectif ;
- CTA CV ;
- CTA GitHub / LinkedIn.

### À propos

Objectif : présenter ton parcours avec sérieux et personnalité.

Axes :

- progression ;
- autonomie ;
- implication ;
- projets réels ;
- capacité d’apprentissage.

### Compétences

Catégories :

- front-end ;
- back-end ;
- base de données ;
- outils ;
- qualité ;
- déploiement.

### Projets techniques

Objectif : montrer les preuves de compétence.

Pour chaque projet :

- contexte ;
- stack ;
- fonctionnalités ;
- lien GitHub ;
- lien démo ;
- points techniques importants.

### Parcours

Objectif : rendre ton profil lisible.

Éléments :

- formation RNCP ;
- projets ;
- expériences ;
- réseau professionnel ;
- objectifs.

### Contact professionnel

Objectif : permettre une prise de contact simple.

Informations :

- email ;
- LinkedIn ;
- GitHub ;
- CV ;
- localisation générale.

---

## Règles éditoriales

- Le portfolio freelance parle au client.
- Le portfolio emploi parle au recruteur.
- Ne pas mélanger les deux tons.
- Éviter les phrases vagues.
- Montrer des preuves concrètes.
- Mettre en avant les résultats, pas seulement les technologies.
- Garder un ton professionnel, humain et premium.
